## WhatsApp 2.0 — Secure Messenger with Moderation

**WhatsApp 2.0** — это десктопный мессенджер с графическим интерфейсом на PySide6, поддерживающий личные и групповые чаты, автоматическую фильтрацию нецензурной лексики, панель модерации для администратора и систему тикетов поддержки.

## 🚀 Функционал

- Регистрация и аутентификация пользователей (пароли хешируются через PBKDF2).
- Личные сообщения между пользователями.
- Групповые чаты: создание группы, вступление по коду-приглашению.
- Автоматическая модерация: сообщения с матом (настраиваемый список) уходят на проверку администратору.
- Панель администратора: просмотр очереди сообщений, одобрение/удаление, ответы на тикеты поддержки.
- Система поддержки: пользователи создают тикеты, администратор отвечает и закрывает их.
- Хранение данных в SQLite (пользователи, сообщения, группы, тикеты).
- Тёмная тема в стиле WhatsApp.
- Автообновление списка сообщений каждые 3 секунды.

## 🛠️ Технологии

- Python 3.10+
- PySide6 (Qt для Python)
- SQLite3
- Модули: `hashlib`, `secrets`, `re`, `dataclasses`

# Parking Lot Manager — Учёт автомобилей на стоянке

**Parking Lot Manager** — это приложение на Tkinter для управления парковкой. Позволяет добавлять машины, фиксировать их заезд и выезд, изменять скорость, останавливать/запускать движение, а также возвращать машину в режим ожидания. Ведётся статистика и автоматическое сохранение в JSON.

## 🚀 Функционал

- Добавление автомобиля (госномер, тип, начальная скорость).
- Редактирование скорости выбранной машины.
- Кнопки **СТОП** (обнуление скорости с сохранением предыдущей) и **СТАРТ** (восстановление скорости).
- **ЗАЕХАТЬ** — фиксирует время въезда на парковку.
- **ВЫЕХАТЬ** — фиксирует время выезда.
- **В ОЖИДАНИЕ** — возвращает машину из состояний `parked` или `left` обратно в `waiting` (сброс времени заезда/выезда).
- Статистика: общее число машин, сколько на стоянке, уехало, ожидает; средняя скорость, количество остановленных и машин с нулевой скоростью.
- Автосохранение в `parking_lot.json` при закрытии программы.
- Интуитивный интерфейс на Tkinter.

## 🛠️ Технологии

- Python 3.7+
- Tkinter (встроенный)
- JSON для хранения данных
- Модули: `uuid`, `datetime`, `os`
